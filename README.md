## Running this app

1. Clone this repository

```sh
git clone https://github.com/MateuszPerczak/hardwareinfo
```

2. Install all dependencies

```sh
pnpm i
```

3. Start react development server and app

```sh
pnpm dev
```

OR if you want to build it

```sh
pnpm build
```
